## ✨ Features

### 🍭 Gameplay
- **100 levels** — progressive difficulty, more gem colours, locked tiles, increasing score targets
- **8×8 board** — swap adjacent gems to match 3 or more in a row or column
- **Combo system** — chain reactions multiply score: 2×, 3×, 4× combo multiplier
- **Hint system** — automatically highlights a valid move after 4 seconds

### 💥 Special Gems (created by matching 4+)
| Gem | How to create | Effect |
|---|---|---|
| 💥 Bomb | Match 4 in a row | Explodes 3×3 area |
| ⚡ Stripe | Match 5 in a row | Clears entire row or column |
| 🌀 Wrap | L-shape or T-shape match | Double explosion blast |
| 🌈 Color bomb | Match 5+ | Clears all gems of one colour |

### 🔨 Boosters
| Booster | Effect | Starting count |
|---|---|---|
| 🔨 Hammer | Remove any single gem | 3 |
| 🔀 Shuffle | Randomise the entire board | 2 |
| 🌈 Color bomb | Remove all gems of one colour | 2 |
| ⏱️ +5 Moves | Add 5 more moves | 1 |

### ✨ Visuals
- Animated particle background (floating sparkles)
- Dark purple theme `#1a0533 → #2d1254`
- Smooth pop/fall animations
- Score float animations
- Combo display overlay

### 💾 Progress Saved
- localStorage: unlocked levels, best scores, star ratings
- 1–3 stars per level based on score
- Level map (tap "Map" to navigate)
